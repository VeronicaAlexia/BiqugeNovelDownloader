

# 基于笔趣阁安卓APP接口小说下载器

---
## 实现功能

| 功能              | 是否实现
|-----------------| -------- |
| 多线程             | ✅       | 
| 异步下载            | ✅       | 
| 命令行操作           |  ✅      | 
| 通过book id下载小说   | ✅      |
| 通过书名下载小说        | ✅      |
| 通过分类序号批量下载小说    | ✅      |
| 通过本地book id批量下载小说 | ✅      |
| 支持epub格式下载      | ✅      |
